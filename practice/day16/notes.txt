## 웹 크롤링 Web Crawling
- 웹 크롤링 행위 자체는 불법이 아니다
- 웹 크롤링의 수행 프로세서를 실제로 살펴보면
    브라우저를 자동적으로 저장하여 데이터를 웹사이트에 요청하고
    요청내용에 해당하는 데이터를 웹 사이트로부터 내려받게 된다

request : 사용자가 서버측에 자료 요청
response : 사용자 요청에 따라 자료(정보)를 응답
사용자 : web browser와 비슷

1. compiler : 소스 프로그램을 기계어로 바꾸어주는 컴파일을 담당한느 번역기
2. interpreter : 번역없이 바로 사용할 수 있도록 라인을 번역하여 실행해주는 역할, 해석기
3. parser : 문자열에 대한 부분을 분리해주는 역할

robots.txt
검색 로봇에게 사이트 및 웹페이지 수집을 혀용하거나 제한하는 국제 권고안
User-Agent : 규칙이 적용되는 대상 사용자 에이전트 정보
Allow : 크롤링 웹페이지를 허용
Disallow : 크롤링 웹페이지를 금지
