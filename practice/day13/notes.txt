# dataframe1
1. pd.concat([파일1, 파일2])
    - 파일을 합치게되면 기본 인덱스를 그대로 가져오기 때문에 이를 처리할 마무리가 필요하다.
    1) df.reset_index(drop=True) : 기본으로 가지고 온 인덱스를 재조정하여 새로운 인덱스로 설정
    2) 파일을 합치게 될 경우에 바로 ignore_index=True에 대한 옵션을 지정하게 되면 인덱스 재조정 없이 바로 새로운 인덱스로 진행
        * inplace=True 없이 바로 사용(알아서 자동으로 인덱스 재조정)

        # 응용
        1. 열 '부가세율'을 삽입 후 0으로 설정
        2. 부가세율을 3%인 0.03으로 재종하여 삽입
        3. 부가세액을 판매금액 * 부가세율을 곱하여 계산
        4. 실제판매가 = 판매금액 + 부가세액을 더하여 계산

# dataframe2
2. pd.merge(파일1, 파일2, on="키값", how="inner") : 키값이 일치되는 파일을 하나로 병합
    이때 기준이 되는 파일을 왼쪽에 위치한다.

    * how option
    inner: 키값이 일치하는 값만 병함
    left : 왼쪽 데이터프레임을 기준으로 병합, 만약 값이 없으면 NA 표시
    right : 오른쪽 데이터프레임을 기준으로 병합, 만약 값이 없으면 NA 표시

    # 응용
    1. 지점별로 그룹화하여 숫자열에 대한 합계를 구하시오
    2. 지점별 세탁기, 에어컨, 냉장고에 대한 평균을 구하시오
